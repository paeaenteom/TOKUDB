# 特撮 ARCHIVE

토에이(東映)·츠부라야(円谷) 특촬 시리즈를 정리한 **단일 HTML 카탈로그**입니다. 가면라이더 · 슈퍼전대 · 메탈히어로 · 울트라 시리즈 등의 작품 / 영웅 / 무장 / 메카 / 괴수 / 조직 데이터를 한 페이지에서 탐색할 수 있습니다.

> 팬이 만든 비영리 아카이브입니다. 이미지·명칭 등의 저작권은 각 권리자(토에이, 츠부라야 프로덕션 등)에게 있습니다.

## 바로 보기

`index.html`을 브라우저로 열면 됩니다. 별도 서버·빌드 없이 더블클릭만으로 동작합니다 (이미지가 파일 안에 포함된 자기완결 HTML).

## GitHub Pages로 공개하기

1. 이 저장소를 GitHub에 올립니다.
2. 저장소 **Settings → Pages** 로 이동합니다.
3. **Source** 를 `Deploy from a branch` 로, 브랜치를 `main` / 폴더를 `/ (root)` 로 지정하고 저장합니다.
4. 잠시 후 `https://<사용자명>.github.io/<저장소명>/` 에서 사이트가 열립니다.

`index.html`이 루트에 있으므로 Pages가 자동으로 첫 화면으로 띄웁니다.

## 현재 수록 현황

- **상세 입력 완료**: 울트라Q(괴수 34종), 초대 울트라맨(영웅 3 · 무장 10 · 메카 9 · 괴수 58 · 조직 1)
- **작품 카드**: 토에이 + 츠부라야 전 작품의 기본 카드(제목 · 연도 · 모티프)
- 나머지 시리즈(슈퍼전대 · 가면라이더 등)는 순차적으로 상세 데이터를 채워가는 중입니다.

## 구조

- 단일 `index.html` 안에 모든 데이터(`DATA` 객체)와 이미지(base64)가 포함되어 있습니다.
- 홈 → 시리즈 → 작품 상세(영웅 / 무장 / 메카 / 괴수 / 조직 / 정보 탭) → 항목 클릭 시 팝업 구조입니다.
- 영웅 이미지는 공통 규칙(동일 캔버스에 인물 중앙 배치)으로 정렬되며, 폼 변경·다른 모습은 토글 버튼으로 전환됩니다.

## 라이선스 / 저작권

이 저장소의 **코드(HTML/CSS/JS)** 는 자유롭게 사용·수정할 수 있습니다.

수록된 **이미지와 캐릭터·작품 명칭**은 각 저작권자(토에이, 츠부라야 프로덕션 등)의 자산이며, 본 아카이브는 비영리 팬 프로젝트로서 자료 정리·소개 목적으로만 사용합니다. 권리자의 요청이 있을 경우 해당 자료를 즉시 수정·삭제합니다.
